# eCommerce Website
The website structure is made for selling beer.

## How To
  - Clone the repository
  ```txt
  git clone
  ```
  - Install the dependencies
  ```txt
  npm install
  ```
  - Run json server with script(modify server script to your liking)
  ```txt
  npm run server
  ```
  - Run npm server
  ```txt
  npm start
  ```

## Tech stack
  - React
  - Symfony
